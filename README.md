# DP Generator

Display Picture Generator - A Next.js app for generating themed display pictures.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- Upload your photo
- Choose from themed design templates
- Add your name to the DP
- Download your personalized display picture
- Share on WhatsApp or Instagram

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Canvas API for image compositing
